# OTOS Continuity — Final Partner Portal Package

This is a static HTML/CSS site package.

## Upload method
1. Create a new branch in your repo.
2. Copy everything in this folder into the repo root.
3. Replace existing files when asked.
4. Commit and push.
5. Check the Vercel preview before merging.

## Simple terminal commands
```bash
git checkout -b final-partner-portal-prepilot
# copy these files into the repo folder, then:
git add .
git commit -m "Build final pre-pilot partner portal"
git push -u origin final-partner-portal-prepilot
```

## Image system
Every page has exactly three image slots:
- `pagename-hero1.png`
- `pagename-image1.png`
- `pagename-image2.png`

Replace image files inside `/images/` using the same names. Do not edit HTML just to change images.

## Site purpose
The site is designed to secure light partner openness to discuss a small, funded pre-pilot demonstrator. It is not a full pilot ask, not procurement, and not a blame document.
