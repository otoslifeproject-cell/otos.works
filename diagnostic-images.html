OTOS image diagnostic

This checks whether the image files are actually loading from the branch. If images show here but not on the site pages, the issue is CSS/layout. If images fail here, the upload/path/name is the issue.

Upload this file to the root of the GitHub branch, next to index.html. Then open /diagnostic-images.html on the Vercel preview. Red outline = missing/broken image. Green outline = image loaded.
