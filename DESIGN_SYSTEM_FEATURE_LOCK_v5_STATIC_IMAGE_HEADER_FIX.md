# OTOS Design System Feature Lock v5 — Static Images + Scrolling Content

Rule now locked:

- Image layers should feel visually static.
- Text and transparent/glass content frames scroll normally above the image layer.
- Do not use sticky hero text.
- Do not hide images behind negative z-index layers.
- Do not convert the whole hero into a sticky block.
- Keep OTOS watermark treatment subtle and low-opacity.
- Keep the top navigation/header fixed and above all image layers.

This patch is intentionally CSS-first and small.
